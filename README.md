# Customer Behavior Analysis Dashboard

This project presents an end-to-end Customer Behavior Analysis solution using Python, SQL Server, and Power BI. The goal of this project is to analyze customer purchasing behavior, revenue trends, customer segmentation, subscription impact, and sales performance through interactive visualizations and data-driven insights.

## Project Workflow

### 1. Exploratory Data Analysis (EDA)
- Performed data cleaning and exploratory data analysis using Jupyter Notebook
- Identified trends, missing values, customer patterns, and category-wise performance
- Used Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn

### 2. SQL Analysis (SSMS)
- Imported cleaned data into SQL Server Management Studio (SSMS)
- Wrote SQL queries for:
  - Customer segmentation
  - Revenue analysis
  - Discount analysis
  - Sales aggregation
  - Category performance analysis

### 3. Power BI Dashboard
Created an interactive Power BI dashboard featuring:
- Total Customers
- Average Purchase Amount
- Average Review Rating
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Subscriber Status Analysis
- Interactive Filters and Slicers

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- SQL Server (SSMS)
- Power BI
- DAX
- Data Visualization

## Key Insights
- Clothing category generated the highest revenue and sales
- Majority of customers were non-subscribers
- Young adults contributed the highest revenue
- Subscription status influenced purchasing behavior
- Revenue trends varied significantly across categories
